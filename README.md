# Java To-Do List App

A simple console-based To-Do List application using Object-Oriented Programming principles in Java.

## Features
- Add new tasks
- View all tasks
- Mark tasks as done
- Remove tasks

## How to Run
1. Clone the repository:
   git clone https://github.com/your-username/project.git
2. Compile:
   javac *.java
3. Run:
   java Main

## Requirements
- Java 8 or later
